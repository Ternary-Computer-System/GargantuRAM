# GargantuRAM

GargantuRAM is a memory module designed for ternary computing architectures.  
It is part of the **Ternary Computer System** initiative, focused on developing hardware and software components for non-binary computation.

## Key Features
- Memory architecture optimized for ternary logic.
- Modular and easily integrable design.
- Documentation and source released under the CERN-OHL-P v2 license.

## Repository Structure
- `src/` — Source code of the module.
- `docs/` — Technical documentation.
- `tests/` — Validation and test materials.
